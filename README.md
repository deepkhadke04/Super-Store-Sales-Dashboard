# Super Store Sales Dashboard

## Overview
The Super Store Sales Dashboard is a data analytics project built using Microsoft Excel and Power Query to analyze sales performance across different regions, categories, and time periods.

The project focuses on transforming raw sales data into interactive visual insights using dashboards, KPIs, and slicers to support business decision-making.

<img src="https://github.com/deepkhadke04/Super-Store-Sales-Dashboard/blob/b917cc0e0c94b43f2fe56bdf9ef551ce79877f13/Dashboard.png" width=600/>

## Objectives
- Analyze overall sales performance
- Identify top-performing regions and cities
- Understand category-wise and segment-wise sales trends
- Build interactive dashboards for business insights

## Dataset
The dataset contains sales records including:
- Order ID
- Customer Segment
- Region
- City
- Category
- Sales
- Quantity
- Shipping Mode
- Order Date

## Key Features
- Dynamic KPI cards
- Interactive filters and slicers
- Region-wise and category-wise analysis
- Monthly sales trend visualization
- Top performing cities identification

## Dashboard Insights
The dashboard includes:

- **Total Sales:** 5,820,182
- **Total Transactions:** 9,994
- **Total Quantity Sold:** 25,123
- **Average Sales per Transaction**


Visualizations included:
- Ship Mode Sales Distribution
- Region-wise Transaction Analysis
- Monthly Quantity Trends
- Top 5 Cities by Sales
- Category-wise Sales Distribution
- Customer Segment Analysis

## Data Processing
Power Query was used to:
